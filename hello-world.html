Hello World

February 07, 2017

Hello world, welcome to my blog! Here is a Fibonacci sequence generator in Python, a test of the syntax highlighter Pygments:

def fibonacci(n):
a = 0
b = 1
for i in range(0, n):
    temp = a
    a = b
    b = temp + b
return a

# Display the first 15 Fibonacci numbers.
for c in range(0, 15):
    print(fibonacci(c))
